This is the environment setup for appium. Please, install the following tools one by one. The version is mentioned as used in the project.

1. install android sdk
2. install jdk 1.8
3. setup system variables for sdk, jdk - (android home & path variables).
4. install python 3.10.4
5. set system environment variables for python
6. install npm 6.14.17
7. setup system environment variables for npm if needed
8. install pip 22.0.4
9. install node.js v16.16.0
10. install appium-python-client
11. install Appium Desktop
